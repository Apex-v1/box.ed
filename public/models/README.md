# 3D Model Files

Drop optimized GLB files here. The model loader (`src/modelLoader.js`) registers
which filenames map to which item types — see `MODEL_PATHS` in that file.

For acquiring and preparing models, see [`docs/MODEL_IMPORT_GUIDE.md`](../../docs/MODEL_IMPORT_GUIDE.md).

This folder ships empty by default so the app works out-of-the-box with
procedural geometry. Add models incrementally as you acquire them.
